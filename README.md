# oneway_anova_cortisol_study
## Overview
A course work project doing statistical analysis on simulated population. Conducted in course STAT1301 at UQ. This project uses simulated experimental data to mimic real-world conditions and demonstrate statistical modelling and hypothesis testing.
## My contribution
- Led the end-to-end statistical analysis
- Visualizing data with result interpretation
- Wrote the main analytical section of the report
- Collaborated with team members (Introduction and sample collections done by peer)
## Contributors
- Joey Yuan

## Dataset
- 57 subjects
- Variables:
  - Treatment (Milk,Coffee,Tea)
  - Salivary
  - Gender
  - Age

## Tools
Used R for analysing and visualizing
## Methodology
- One-way ANOVA
- Assumption checks:
   - Normality (Q-Q plot)
   - Homogeneity of variance
## Results
- Significant difference between groups (p < 0.000218)
